PEA METER GUIDE
คู่มือการดูข้อมูลงานจ่ายมิเตอร์
เว็บนี้ใช้สำหรับดูข้อมูลงานจ่ายมิเตอร์ของพื้นที่บางละมุง ช่วยให้พนักงานติดตามงานได้ง่ายขึ้น และช่วยให้ผู้ใช้ไฟเข้าใจสถานะ ประเภทงาน และขั้นตอนที่เกี่ยวข้องกับงานมิเตอร์

👤 สำหรับผู้ใช้ไฟ

  • ค้นหางานจากเลขคำร้อง ชื่อ ที่อยู่ หรือเลขมิเตอร์
  • ดูสถานะว่างานรอดำเนินการหรือปิดงานแล้ว
  • ทำความเข้าใจประเภทงาน เช่น ติดตั้งใหม่ เพิ่มขนาด TOU
  • ดูตำแหน่งงานโดยรวมผ่านหน้าแผนที่

👷 สำหรับพนักงาน

  • ตรวจสอบรายการงานค้าง
  • ดูสรุปงานรายวันหรือรายสัปดาห์
  • กรองงานตามผู้ปฏิบัติงาน ประเภทงาน และ Zone
  • ใช้แผนที่ช่วยวางแผนพื้นที่ปฏิบัติงาน

🔎 ค้นหาได้จากอะไรบ้าง

สามารถค้นหาข้อมูลงานได้จากคำสำคัญหลายแบบ

เลขคำร้อง ชื่อผู้ใช้ไฟ เลขมิเตอร์ PEA ที่อยู่ MRU ผู้ปฏิบัติงาน

🏷️ ประเภทงานที่พบในระบบ

  • ติดตั้งใหม่ งานขอใช้ไฟและติดตั้งมิเตอร์ครั้งแรก
  • เพิ่มขนาด งานเปลี่ยนขนาดมิเตอร์ให้รองรับการใช้ไฟมากขึ้น
  • เปลี่ยนประเภท งานเปลี่ยนประเภทการใช้ไฟตามลักษณะการใช้งาน
  • TOU งานมิเตอร์อัตราตามช่วงเวลาการใช้ไฟ
  • ต่อกลับ งานต่อไฟกลับหลังจากถูกงดจ่ายหรือถอดมิเตอร์

📌 ความหมายสถานะงาน

  • รอดำเนินการ งานที่ยังอยู่ระหว่างตรวจสอบ รอติดตั้ง หรือรอบันทึกผล
  • ปิดงาน งานที่ดำเนินการเสร็จและบันทึกผลเรียบร้อยแล้ว

🧭 เมนูหลักในระบบ

  • ค้นหางาน ค้นหาและดูรายละเอียดงานแต่ละรายการ
  • งานค้าง ดูงานที่ยังรอดำเนินการ
  • สรุปรายวัน ดูภาพรวมงานตามวันที่
  • แผนที่ ดูตำแหน่งงานตามพื้นที่
  • Dashboard ดูภาพรวมจำนวนงาน
หมายเหตุ: หน้านี้เป็นคู่มือการใช้งานและคำอธิบายระบบเท่านั้น ไม่ได้แก้ไขข้อมูลใน Google Sheet ข้อมูลจริงยังมาจากแหล่งข้อมูลที่ตั้งค่าไว้ในระบบเดิม
📄 คู่มือ PDF จากภายนอก
เพิ่มไฟล์ PDF จากเครื่อง แล้วเปิดอ่านในเว็บได้ทันที ระบบรับเฉพาะไฟล์ .pdf เท่านั้น
ยังไม่มีไฟล์ PDF ในคู่มือ
🔍 ค้นหางานจ่ายมิเตอร์
ใช้หน้านี้เพื่อตรวจสอบรายการงานจ่ายมิเตอร์ ดูสถานะ ประเภทงาน พื้นที่ Zone และรายละเอียดผู้ใช้ไฟ กดที่รายการในตารางเพื่อดูข้อมูลเพิ่มเติมหรือพิมพ์ใบงาน
เลขคำร้องชื่อผู้ใช้ไฟเลขมิเตอร์ PEAที่อยู่MRUผู้ปฏิบัติงาน

🏷️ ประเภทงานที่พบบ่อย

  • ติดตั้งใหม่ ขอใช้ไฟและติดตั้งมิเตอร์ครั้งแรก
  • เพิ่มขนาด เปลี่ยนขนาดมิเตอร์ให้รองรับโหลดมากขึ้น
  • เปลี่ยนประเภท เปลี่ยนลักษณะการใช้ไฟตามการใช้งาน
  • TOU มิเตอร์อัตราตามช่วงเวลาการใช้ไฟ
  • ต่อกลับ ต่อไฟกลับหลังจากถูกงดจ่ายหรือถอดมิเตอร์

📌 ความหมายสถานะงาน

  • รอดำเนินการ งานที่ยังอยู่ระหว่างตรวจสอบ รอติดตั้ง หรือรอบันทึกผล
  • ปิดงาน งานที่ดำเนินการเสร็จและบันทึกผลเรียบร้อยแล้ว
  • ไม่มีพิกัด รายการที่ยังไม่มี Latitude/Longitude สำหรับแสดงบนแผนที่

🔗 Auto-sync จาก Google Sheet
วาง URL ที่ได้จาก Apps Script แล้วกด บันทึก — ระบบจะดึงข้อมูลสดทุกครั้งที่เปิดเว็บ
📋 วิธีติดตั้ง (ทำครั้งเดียว 5 ขั้นตอน)
1 เปิด Google Sheet → เมนูบน ส่วนขยาย → Apps Script
2 ลบโค้ดเดิมทิ้ง → วางโค้ดจากไฟล์ Code.gs ที่ได้รับ → กด 💾 บันทึก
3 กด Deploy → New deployment → ประเภท Web app
4 ตั้งค่า Execute as: Me · Who has access: Anyone → กด Deploy → อนุญาต permission
5 คัดลอก Web app URL → วางในช่องด้านบน → กด บันทึก
หมายเหตุ
• URL จะบันทึกใน browser นี้ ไม่ต้องตั้งค่าซ้ำ
• เว็บนี้ใช้ข้อมูลสดจาก Google Sheet เท่านั้น ไม่มีข้อมูลเก่าฝังในไฟล์
• ถ้าอัปเดต Apps Script ต้อง Deploy เวอร์ชันใหม่ทุกครั้ง
